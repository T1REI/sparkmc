# sparkmc (Rust)

Native console edition of **sparkmc**.

## Build

```bash
cargo build --release
```

Binary:

```text
target/release/sparkmc       # Linux / macOS
target/release/sparkmc.exe   # Windows
```

## Run

```bash
./target/release/sparkmc
```

```powershell
.\target\release\sparkmc.exe
```

## Requirements

- [Rust](https://rustup.rs/) stable
- Network for downloads

See the [root README](../README.md) for shared features and `sparkmc.json` format.
